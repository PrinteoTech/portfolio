Projects

B.A.R.Y.O.N

A miniature tokamak for energy.

B.A.R.Y.O.N

The aim of the BARYON project is to design a miniature tokamak capable of producing enough energy to power a house using nuclear fusion. I carried out hand calculations to model plasma behavior, magnetic confinement and reactor design. I worked on modeling the tokamak's geometry, analyzing the magnetic fields required and simulating the optimal configuration for its operation. This project enabled me to develop my skills in research, technical design, physics and complex problem-solving.

A.T.H.E.N.A

An AI evolving into an assistant.

A.T.H.E.N.A

The ATHENA project aims to create an artificial intelligence capable of functioning as an intelligent assistant, like JARVIS, to process data and interact autonomously. I designed and trained a neural network using supervised learning techniques to analyze data and improve accuracy over time. During the test phase, I refined the model to make it more responsive and efficient. This project has enabled me to strengthen my skills in AI and neural networks.
